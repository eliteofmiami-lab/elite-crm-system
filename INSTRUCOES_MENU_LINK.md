# Daily Board DENTRO do GHL — Custom Menu Link (3 minutos, uma vez)

O painel diário (espelho do GHL) vive numa página completa, feita pra abrir dentro
do GHL.

## Passos

1. No GHL, abra **Settings** (engrenagem no menu esquerdo, embaixo).
2. Clique em **Custom Menu Link** (em *Company/Location settings*).
3. **+ Add Custom Menu Link**:
   - **Title:** `Elite — Daily Board`
   - **URL:** `https://elite-crm-panel.vercel.app/`
   - **Icon:** 📋 (ou o que preferir)
   - **Open in:** `iFrame` (abre DENTRO do GHL). Se ficar em branco, use `New tab`.
   - **Show to:** location Elite (Ao5ER8XBg3AtCJMccesF), visível pra você e Eugene.
4. Salvar → o item aparece no menu lateral do GHL.

## Primeiro uso

- Login com o e-mail/senha de sempre do painel (cada um o seu). Fica logado.
- **Eugene:** o quadro só libera depois do **Clock in** (botão na própria tela).
- **Você:** além do Board, tem a aba **Owner · Rafael** (contadores, sem-resolução,
  appointments com notas, ganhos do Eugene, beta feedback).

## O que é (e o que não é)

- **Espelho do GHL a cada 5 min** — calls, mensagens, stages, tasks, appointments.
- **Nenhum botão escreve no GHL.** Todo registro (stage, task, nota, Lost) continua
  sendo feito no próprio GHL — o painel detecta e os cards se fecham sozinhos.
- **Regra da Resolução:** discagem não fecha card; desfecho fecha (appointment ·
  task com data · estimate + Quote Sent · Lost com motivo; não atendida → próximo
  stage). Call sem desfecho em 15 min = card vermelho **NO RESOLUTION**.
- **Card verde:** 100 ligações válidas/dia, escada 30/35/40 ($300/$400/$500),
  potencial pendurado nas confirmações, bônus de quadro limpo $50/quinzena
  (teto $600/mês).
